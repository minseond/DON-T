# 📌 오늘 작업 기록 (2026-03-04)

## 1) 유저 플로우(User Flow) 설계
- 금융 도메인 주요 플로우 정리
  - 카드 연동 → 거래내역 수집/표준화 → AI 지출 분석 리포트 노출 흐름 구성
  - 연동 상태(정상/만료/재인증 필요/외부 장애), 수집 상태(정상/지연/일부 누락) 등 분기/예외 케이스 반영
- 랭킹 도메인 유저 플로우 정리
  - 유저 조회 플로우(랭킹 탭 → 내 그룹/기수 스코프 → 최신 스냅샷 조회 → 티어/강등예정 표시)
  - 시스템 배치 플로우(하루 1회 점수 산정 → 순위/티어/강등예정 → 스냅샷 저장/이벤트 발행)
- 커뮤니티 플로우 보강
  - 목록/상세/작성/댓글/반응/신고 흐름 정리
  - 커뮤니티 진입 시 카테고리 탭(전체/기수별/구매PR)로 분기되는 구조 반영

## 2) 기능 명세서(Functional Spec) 정리 및 기능 설명 작성
- 사용자/인증
  - 회원가입, 이메일 인증, 로그인/로그아웃, 로그인 상태 확인
  - 아이디 찾기, 비밀번호 재설정
  - 온보딩 입력/수정, 추천 저축금액 산정, 목표 저축금액 저장(추천값과 분리 저장)
  - 익명 닉네임 자동 생성 및 변경(유효성/금칙어/중복 정책 포함)
- 금융
  - 금융 동의 수집/이력 관리(버전/시각 포함)
  - 입출금 통장 연동(주거래 통장 등록/변경, 재인증 필요 상태 반영)
  - 카드 연동/관리(사용내역 수집 대상 등록, 만료/재인증 상태 관리)
  - 세이브 박스 개설 및 상태 관리
  - 거래내역 수집 및 표준화/중복 제거
  - 월급 감지 이벤트 생성 및 자동저축 트리거 전달
  - 자동저축 금액 산정(목표-누적 기반) 및 자동이체 실행/기록
  - 자동저축 실패 분류 및 재시도 큐 처리
  - 세이브 박스 잔액/거래내역 조회(자동저축/이자/전체 필터)
  - 월 지출 요약/카테고리 집계/과소비 신호 생성
  - 일평잔 기반 이자 계산 및 반영(중복 방지)
  - 중복 방지 및 정합성 보정(재집계 포함)
  - 연동 만료/권한 오류/외부 장애 상태 관리 및 복구
- 랭킹
  - 그룹/기수/사용자 배정 관리
  - 점수 산출(입금+이자-출금), 일일 갱신 배치, 랭킹 조회
  - 티어 산정(상위 퍼센트 컷), 강등 예고 판정
  - 스냅샷 저장(그룹+기수+일자) 및 이벤트 발행
- 커뮤니티/구매PR
  - 커뮤니티 범위(그룹/기수) 및 익명 닉네임 노출 규칙
  - 목록 페이징/검색/정렬
  - 게시글/댓글 CRUD 및 권한 정책
  - 반응(좋아요/싫어요) 멱등 처리
  - 신고(게시글/댓글) 누적 임계치 기반 블라인드/검토대기 처리
  - 구매PR 생성 → 리뷰(승인/반려) → 최종결정(MERGE/REJECT) 흐름
