# 기획 배경

SSAFY 교육생은 한정된 지원금과 생활비 안에서 소비와 저축을 스스로 관리해야 하지만, 단순 가계부 서비스만으로는 지속적인 동기부여가 어렵다고 판단했습니다.  
이에 DON'T는 단순한 소비 기록 서비스를 넘어, 저축과 소비 관리를 **커뮤니티**, **랭킹**, **실시간 알림**, **AI 소비 분석**과 결합하여 건강한 금융 습관 형성을 돕고자 기획되었습니다.

특히 비슷한 경제 환경을 가진 교육생들이 서로의 경험을 공유하고, 구매 전 의견을 묻거나 저축 성과를 확인할 수 있도록 구성하여 금융 관리의 재미와 지속성을 높이고자 했습니다.

# 서비스 화면 구성

## 1. 메인 / 홈
- 사용자의 주요 금융 현황 확인
- 저축 현황 및 서비스 핵심 기능 진입

## 2. 스마트 세이브박스
- 사용자가 설정한 저축률에 따라 세이브박스에 자금 저축
- 자동/수동 저축 내역 확인
- 저축 현황 시각화

## 3. 랭킹 시스템
- 비슷한 사용자 간 저축률 비교
- 실시간 랭킹 확인을 통한 동기부여 제공

## 4. 커뮤니티
- 소비 경험 공유
- 구매 PR 게시글 작성
- 댓글/답글을 통한 사용자 간 소통

## 5. 알림
- 댓글, 답글, 신고 처리, 저축 이벤트 발생 시 실시간 알림 제공
- 관련 콘텐츠로 바로 이동 가능

## 6. AI 소비 분석
- 월간 소비 패턴 분석
- 개인 맞춤형 저축 및 소비 개선 방향 제안

# 주요 기능

## 1. 스마트 세이브박스
- 맞춤형 저축 설정
- 자동/수동 저축 기능
- 저축 내역 및 현황 확인

## 2. 저축 랭킹
- 사용자 간 저축률 비교
- 실시간 순위 확인
- 건강한 경쟁을 통한 저축 동기부여

## 3. 구매 PR
- 구매 전 커뮤니티에 의견 요청
- 댓글 및 피드백을 통한 합리적 소비 유도

## 4. 커뮤니티
- 게시글, 댓글, 답글 작성
- 사용자 간 소비 경험 공유
- 신고 기능 제공

## 5. 실시간 알림
- 댓글/답글 작성 시 알림
- 신고 처리 결과 알림
- 세이브박스 저축 이벤트 알림
- SSE 기반 실시간 알림 제공

## 6. AI 소비 분석
- 소비 패턴 분석
- 맞춤형 피드백 제공
- 월간 리포트 생성
