## 🎯 기획 배경

SSAFY 교육생은 한정된 지원금과 생활비 안에서 소비와 저축을 스스로 관리해야 하지만,  
단순한 가계부 서비스만으로는 **지속적인 동기부여**를 만들기 어렵다고 생각했습니다.

**DON'T**는 단순한 소비 기록 서비스를 넘어,  
✅ **저축 랭킹**  
✅ **구매 PR**  
✅ **AI 소비 분석**  
✅ **실시간 알림**  
✅ **커뮤니티 기능**  
을 결합하여 **건강한 금융 습관 형성**을 돕고자 기획되었습니다.

특히 비슷한 환경의 교육생들이 서로의 경험을 공유하고,  
구매 전 의견을 묻거나 저축 성과를 확인할 수 있도록 구성하여  
금융 관리가 더 이상 혼자만의 일이 아니라 **함께 만들어가는 경험**이 되도록 설계했습니다.

## ✨ 주요 기능

### 💰 1. 스마트 세이브박스
- 사용자가 설정한 저축률에 따라 **자동/수동 저축**
- 저축 내역 및 현황 확인
- 계획적인 소비와 저축 습관 형성 지원

### 🏆 2. 저축 랭킹
- 사용자 간 저축률 비교
- 실시간 순위 확인
- 건강한 경쟁을 통한 저축 동기부여

### 🛒 3. 구매 PR
- 구매 전 커뮤니티에 의견 요청
- 댓글 및 피드백을 통한 합리적 소비 유도
- 충동구매 방지와 사용자 간 상호작용 강화

### 💬 4. 커뮤니티
- 게시글 / 댓글 / 답글 작성
- 소비 경험 공유
- 신고 기능 제공

### 🔔 5. 실시간 알림
- 댓글 / 답글 작성 시 알림
- 신고 처리 결과 알림
- 세이브박스 저축 이벤트 알림
- **SSE 기반 실시간 알림 제공**

### 🤖 6. AI 소비 분석
- 소비 패턴 분석
- 맞춤형 피드백 제공
- 월간 리포트 생성
